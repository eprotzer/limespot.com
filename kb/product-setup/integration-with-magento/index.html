Integration with Magento

Importing products and collections from your Magneto store onto LimeSpot is fairly easy and straight forward. The first step is to enable API access for your existing Magneto account and then linking this account with your LimeSpot Admin Panel.

Magento has three types of subscription: Magento GO, Magento CE and Magento EE. While our system works with all subscription types, our services are optimized for Magento GO stores.

Below is a guide to the integration process.

Enabling API Access

Web Services give you the ability to integrate with third-party applications by granting them permission to access specific store resources and data.

Step 1: Create a Web Service Role

  1. From Magento's Admin menu, select System > Web Services > Roles.
  2. Click the Add New Role button.
  3. In the Role Information section, enter a Role Name such as "API."
  4. In the panel on the left, select Role Resources and set Resource Access to "All".
  5. When complete, click the Save Role button.

Step 2: Create a Web Service User

  1. From the Mangento admin menu, select Web Services > Users.
  2. Click the Add New User button.
  3. Complete the following fields:
    • User Name: a username of your choice. eg, limespotAPI.
    • First Name: a name of your choice. eg, limespot.
    • Last Name: a name of your choice. eg, API.
    • Email: a valid email address that you own and manage.
  4. In the API Key field, enter a verification key or password. This is the credential that LimeSpot services use to access your Magento Go store.
  5. To confirm, re-enter the key in the API Key Confirmation field.
  6. In the panel on the left, select User Role. Then, select a Role for the user. (One that you created previously)
  7. When complete, click the Save User button.

Linking Magento Go Store with LimeSpot

  1. Select E-Commerce Connect from the LimeSpot Admin Panel.
  2. Locate the Mangento section on this page.
  3. Enter the username you created in Magento. eg, limespotAPI.
  4. Enter the API Key and your store URL.
  5. If required, change the Product Auto-generated setting. By default this is set to Name. This means that when you create multiple variants of a product, Magento differentiates between variants by their product name. You also have an option of changing the Product Auto-generated setting to SKU, wherein multiple product variants have same name but are differentiated by unique SKUs.
  6. Click on Validate and Connect to establish a connection.
  7. You will receive a confirmation message indicating a successful connection.

Please note that you should select the same Product Auto-generated setting as you did for your Magento Store. If you have not changed this attribute in your Magento store then by default it is always set to Name.

Importing Products into LimeSpot

  1. Click on Product Data Import from the LimeSpot Admin Panel.
  2. Select the button next to your Magento Store's Name.
  3. Click on Import Products in the top right corner of the screen.

Please note that you should click on Import Products to synchronize updates every time you add more products or change product details on your Magento store.
