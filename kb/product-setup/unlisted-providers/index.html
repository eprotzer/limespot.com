Integration with Unlisted E-commerce Providers

We strive to continuously add support for automatic integration of LimeSpot shops with major Ecommerce systems.

Currently we suport the following e-commerce providers:

  1. Shopify
  2. Magento
  3. BigCommerce

Please contact Customer Support if you need assistance for integration with an e-commerce system for which we do not support automatic integration.

Some other alternative methods of adding products onto LimeSpot platform are:

  1. Import Products Manually using Spreadsheets
  2. Manually Create Products

We are also in the process of creating some generic APIs for ecommerce providers to connect with LimeSpot platform.
