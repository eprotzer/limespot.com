Custom Contents

You can use Custom Contents to create HTML Tags enclosing files of various formats such as multimedia (audio/video), Text, Flash, iFrames, etc. Custom Contents can be added to your online shops to enable several types of HTML supported functionalities.

Some of the most notable uses of Custom Contents are:

  • Showing rich graphics files.
  • Displaying videos.
  • Creating forms for user input.
  • Implementing JavaScript.
  • Implementing Creative Animations and Visual Effects.

Custom Contents can be accessed from the Products and Contents section of the Admin Panel. The landing page for Custom Contents looks like this:

From the Landing Page of Custom Contents you can:

  • Create new Custom Contents
  • Copy existing Custom Contents
  • Edit Custom Contents
  • Delete Custom Contents
  • View Custom Content as seen by your customers during Runtime

How to Create Custom Contents

  1. Select the Custom Contents tab in the Admin Panel.
  2. Click on the Add Custom Content button in the top right corner of the screen. You will see the following screen:
  3. Enter a Title for Custom Content you are creating.
  4. Choose an image file that the Custom Content will display.
  5. Enter a description for internal reference. You customers never see the contents of the Description field.
  6. Enter the required HTML code.
  7. Enter relevant Targeting Keywords.
  8. Filter or Prioritize for appropriate Targeting Rules.*
  9. Click on the Save button to complete the process.

* Our system provides a pre-generated list of Targeting Rules that cover most widely used conditions. However, you are free to create your own Targeting Rules from the Admin Panel.

Targeting Rules are added to display personalized custom contents to different customers. For example, by adding a Targeting Rule, you could show two different marketing videos to men and women. The Targeting Rule would be set to identify the user's gender and our engine will display the related video accordingly.
