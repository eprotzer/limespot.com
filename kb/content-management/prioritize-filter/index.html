Prioritizing & Filtering Objects

All Objects including External Content, Custom Content and Products can be associated with Targeting Rules in two ways: Filter and Prioritize.

Whenever you create a new object or edit an existing object you can find a section labeled Targeting Rules at the bottom of the page. This section is populated with all the Targeting Rules that you have previously created. You will notice that each targeting rule has three adjacent click buttons: Filter, Prioritize, and None. Refer to Targeting Rules to learn how to create targeting rules and what they do.

Filter an Object

You should filter an object for a targeting rule when you want to display the object if and only if the related targeting rule 's condition is met.

For example, if one of your products is gourmet dog food and you have a targeting rule to identify dog lovers you should set the targeting rule to 'Filter', since those are the only people who might be interested in purchasing gourmet dog food. Doing so will make the product visible to people if and only if they have indicated fondness for dogs on their social profiles.

Conversely, you might not want to filter a pet blanket for dog lovers as different types of pet owners might be interested in the product.

Prioritize an Object

You should select the Prioritize button for more generic objects that can be related to two or more targeting rules. You can prioritize a product for two or more targeting rules that represent different categories of people.

For example, if you are selling a pet accessory that is used by both cats and dogs you should prioritize the product for Cat Lovers and Dog Lovers. This will make sure that people who like cats or dogs see this pet accessory.

If you feel that a certain product has no relevance with any Targeting Rule then you should choose None option. Objects with no Targeting Rules are shown to all shoppers. However, such objects are displayed with the least priority.

Priority of an Object decides the rank or order in which Objects are displayed on product catalogues. Objects with higher rank are displayed first and vice versa.

Our engine ranks Filtered Products ahead of Prioritized Products, followed by untargeted products.
